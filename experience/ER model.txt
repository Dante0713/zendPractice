ER model 實體關係圖(Entity-Relationship Diagram)
	- Entity <長方形>
		- 一般實體
		- 弱實體 //雙長方形
	- Relationship <菱形>
		- 一對一
		- 一對多
		- 多對多
	- Attribute <橢圓形>
		- 鍵屬性
			- 辨別實例的屬性
		- 弱實體識別屬性
			- 弱實體的鍵屬性 //下底虛線
		- 推導屬性
			- 可由其他屬性推論而得 //虛線
		- 複合屬性
			- 由多個屬性組合而成的 //樹狀分支
		- 多值屬性
			- 多個以上的值 //雙橢圓形
EER model 擴充實體關係模型(Extended Entity-Relationship Model)
	- 超類型 Super Type
		- Interface class
	- 子類型 Sub Type
		- extends class
		- disjoint 不相交關係
			- 非此即彼，不可能同時擁有兩個身分
				- 一位學生身分為大學生時，就不可能同時是一位中學生
		- overlap 重疊關係
			- 多個身分重疊的人
				- 客戶同時是租屋者亦是買屋者
	- 特殊化 Specialization
		- 子類型超類型的動作
	- 歸納化 Generalization
		- 整理多個相似類型，並建立abstract class
